Background:waitForInit (tmo, ival)
Wait for asynchronous initialization to complete.

Parameters:

  • tmo: - initial timeout in seconds - default is half second.
  • ival: - recheck interval if not finished initializing upon initial timeout - return immediately and do not prompt user if nil or zero. Ignored if tmo is zero.

Usage:

    Untested @2011-01-08 ###2

Return values:

  1. status - true iff init completed successfully.
  2. explanation - if status false, is error message. if status true, indicates some waiting was required.

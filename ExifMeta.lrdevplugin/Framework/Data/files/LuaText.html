LuaText:deserialize (s, name, o)
Deserialize previously serialized lua data table from s, and optionally: assign to object o.

Parameters:

  • s: the lua format string to deserialize.
  • name: optional chunk name in case of error.
  • o: optional table to receive deserialized key/values (I can no longer remember the reason for this).

Return value:

    lua table object

Serialize lua data object, typically a table.

Parameters:

  • t: the lua table or simple variable to be serialized.

Usage:

    excludes the dressing needed to use said table upon deserialization, so make sure you precede it with a "return " or "myTbl = " or something.

    serialized lua code or nil (never empty string).

Collections:getCollPath (coll, sep)
Get path to collection or set.

Parameters:

  • coll: (Lr collection, or set, required) can not be nil, but can be a special collection.
  • sep: (string, default: app--path-sep) path separator.

Usage:

    assuming coll is a collection, even if special.

Return values:

  1. path (string) If no parent returns collection name, even if special.
  2. comp (array of strings) components, in case you want to roll your own path or sub-path.. last element is name of collection itself.
